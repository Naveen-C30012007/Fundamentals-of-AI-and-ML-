# Fundamentals-of-AI-and-ML-
# 📊 Loan Repayment Prediction

## 📌 About
This project predicts whether a person will repay a loan or not using basic Machine Learning.

## 🎯 Aim
To build a simple model that helps in identifying:
- Loan will be repaid ✅  
- Loan will not be repaid ❌  

## 📂 Files in Project
- loan_data.csv → Dataset  
- loan-repayment-prediction.ipynb → Code file  

## 🛠️ Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

## ⚙️ Steps
1. Load dataset  
2. Clean the data  
3. Analyze data  
4. Train model  
5. Test model  

## 🤖 Model Used
- Logistic Regression  
- Decision Tree (basic)

## ▶️ How to Run
git clone https://github.com/Naveen-C30012007/Fundamentals-of-AI-and-ML-.git  
cd Fundamentals-of-AI-and-ML-  
jupyter notebook  

## 📈 Output
The model predicts whether a loan is:
- Repaid  
- Not Repaid
